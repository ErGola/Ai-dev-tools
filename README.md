# Ai-dev-tools

This workspace contains a small Snake game built with React + Vite + Tailwind CSS.

Quick start

1. Install dependencies:

```bash
npm install
```

2. Start dev server:

```bash
npm run dev
```

Open http://localhost:5173 in your browser.

Controls: use arrow keys or WASD. Space to pause/resume.
